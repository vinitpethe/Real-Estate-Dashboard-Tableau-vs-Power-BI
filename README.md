# 📊 Real Estate Dashboard Comparison: Tableau vs Power BI

# Project Overview
This project presents a comparative analysis of **Tableau and Power BI**
using the same **real estate sales dataset**.

The objective was to evaluate both tools by building dashboards with
similar visualizations and comparing them on:

- Ease of use
- Data preparation
- Dashboard design
- Interactivity
- Calculated fields
- Overall user experience

The dashboards focus on **property pricing, sales count, monthly trends,
zip code performance, and bedroom-based sales analysis**.

---

## 🎯 Objective
The goal of this project was to determine which BI tool performs better
for real estate dashboard development based on:

- Visual design flexibility
- Data transformation efficiency
- Ease of calculated field creation
- Filter interactivity
- Dashboard layout and usability

---

## 📂 Dataset
The project uses a **real estate sales dataset** containing:

- Property prices
- Square footage
- Bedrooms
- Zip codes
- Property size categories
- Property age (new vs old)
- Monthly sales trends
- Sales grade

---

## 🛠 Tools Used
- **Tableau**
- **Power BI**
- **Excel / CSV Dataset**
- **DAX (Power BI)**
- **Calculated Fields (Tableau)**

---

## 📈 Dashboard Features
### Tableau Dashboard
- KPI cards
- Top zip codes by property sold
- Average adjusted sales price by bedrooms
- Sales by zip code
- Monthly sales trend
- Property category parameters
- Interactive action filters

### Power BI Dashboard
- KPI cards
- Slicers for year and property category
- DAX measures
- Bookmarks
- Cross-page filtering
- Monthly sales line chart
- Donut chart analysis

---

## 🔍 Comparative Analysis
### ✅ Tableau Strengths
- Easier drag-and-drop interface
- Faster calculated field creation
- Better dashboard alignment
- Superior visual customization
- Easier parameter-based filtering
- More flexible design containers

### ✅ Power BI Strengths
- Better built-in data cleaning with Power Query
- Strong DAX capabilities
- Better integration with Excel and Microsoft ecosystem
- Powerful slicers and bookmarks

---

## 💡 Key Insights
- Tableau offered a **more intuitive dashboard-building experience**
- Power BI required **more technical effort due to DAX**
- Tableau made **interactive parameter creation simpler**
- Power BI excelled in **data cleaning and transformation**
- Tableau dashboards were easier to align and visually customize

---

## 🏆 Final Conclusion
After comparing both tools, **Tableau was preferred for this project**
because of its:

- Better visual aesthetics
- Easier interactivity setup
- Simpler calculated fields
- Faster dashboard development workflow

Power BI remains highly effective for business users working within
the Microsoft ecosystem.
